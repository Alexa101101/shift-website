Shift reinvents decentralized cloud hosting with Lock n’ Load™ technology.

Store your files in three simple steps.

  1. Lock

    Start by locking some SHIFT to receive storage

  2. Upload

    Next, select a file to upload

  3. Pin it

    Here’s your hash! Now, record it permanently on the blockchain

  4. Load

    Lift off! Open your file using our gateway

  1. Unpin it

    Select the hash of a file you no longer wish to have saved

  2. Unlock

    Are you done? Unlock your SHIFT

...and when you’re done, retrieve your tokens by reversing these steps.
